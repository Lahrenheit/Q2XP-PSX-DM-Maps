All credits for made these awesome maps!

Relight (.xplm .xplit) and levelshots made by Lahrenheit

All PSX & N64 DM Maps (no Badlands and Overflow)



## Setup

Download the "release" .pkx file and paste to quake2/baseq2 folder

You can create or edit the file "maps.lst" in baseq2 folder and write:
```
q2ps1 "Cold Storage"
q2ps2 "Warehouse"
q2ps3 "Aquaplex"
q2ps4 "Coliseum"
q2ps5 "Mainframe"
q2ps6 "Quickfire"
q2ps7 "The Shaft"
q2ps8 "Hydraphobia"
q2ps9 "Toxic Vats"
toxic "Toxic Vats HD"
q2ps10 "The Forge"
forge "The Forge HD"
q2ps11 "Captivity"
q264_1 "Agony"
q264_2 "Corridors"
q264_3 "War Room"
q264_4 "Twists"
q264_5 "Fortress"
q264_6 "Cold Steel"
q264_7 "Mad Dash"
q264_8 "Empty Space"
q264_9 "Dying Halls"
```
